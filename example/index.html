Puse CSS Framework

Repository Download

Puse is a CSS framework. It's being created to be managable by Duo for faster prototyping of web apps. Source code can be accessed on Github repository. Additional components might appear under organization namespace and will follow same style and philosophy as Puse itself.

As a reference were used framworks:

  • Pure.css
  • Base
Name Puse comes from it's ancestors' names combined.

Puse is built on top of Normalize.css.
